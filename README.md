## Installation

1. Open your terminal or command prompt
2. Clone the project

```bash
$ git clone git@github.com:sipamungkas/flutter-basic.git
```

3. Move inside the directory and install and link dependency

```bash
$ cd flutter-basic
$ flutter pub get
```

4. Run program

```bash
$ flutter run
```
